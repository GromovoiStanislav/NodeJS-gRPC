## Generate JS code from xxx.proto file with protoc-gen-grpc (JS/.d.ts) (TS)

Generating TypeScript Definitions for CommonJS modules generated by protoc:

```
npm install protoc-gen-grpc -g

protoc-gen-grpc --js_out=import_style="commonjs,binary:./src/generated" --grpc_out=grpc_js:./src/generated --proto_path ./src/proto ./src/proto/*.proto
protoc-gen-grpc-ts --ts_out=grpc_js:./src/generated --proto_path ./src/proto ./src/proto/*.proto
```

#### Build:

```
npm run build
```

#### Run:

```
npm run server
npm run client
```
