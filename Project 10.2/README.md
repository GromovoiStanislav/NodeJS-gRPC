## Generate static code with protobufjs-cli (JS)

#### Setup:

```
npm i protobufjs-cli
npm i protobufjs
```

#### Generate static code:

```
npm run bundle-js
npm run bundle-json
```

#### Start:

```
npm run example-1
npm run example-2
npm run example-3
npm run example-4

npm run example-5
npm run example-6
npm run example-7
```
