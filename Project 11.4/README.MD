## Generate JS code from xxx.proto file with ts-protoc-gen (JS/.d.ts)

Generating TypeScript Definitions for CommonJS modules generated by protoc:

```
protoc --plugin=protoc-gen-ts=".\\node_modules\\.bin\\protoc-gen-ts.cmd" --js_out=import_style=commonjs,binary:./generated --ts_out=".\\generated" ./protos/users.proto

protoc --plugin=protoc-gen-ts=.\\node_modules\\.bin\\protoc-gen-ts.cmd --js_out=import_style=commonjs,binary:. --ts_out=. ./protos/users.proto
```

```
protoc --plugin="protoc-gen-ts=./node_modules/.bin/protoc-gen-ts" --js_out="import_style=commonjs,binary:./generated" users.proto
protoc --plugin="protoc-gen-ts=./node_modules/.bin/protoc-gen-ts" --js_out="import_style=closure,binary:./generated" users.proto
```
