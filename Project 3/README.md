# NodeJS gRPC
Just a quick example of a gRPC server made with NodeJS<br>
It contains 4 different examples:
1. Unary
2. Server Streaming
3. Client Streaming
4. Bidirectional Streaming


## Project setup
```
nmp i
```

### Runs the gRPC server
```
nmp run server
```

### Runs out client
```
nmp run client
```
