# NodeJS gRPC and ExpressJS

Just a quick example of a gRPC server made with NodeJS and client ExpressJS<br>
It contains 2 different examples:

1. Unary
2. Server Streaming

### Runs the gRPC server

```
nmp run server
```

### Runs client

```
nmp run client
```

### Runs client express

```
nmp run client-express
```
