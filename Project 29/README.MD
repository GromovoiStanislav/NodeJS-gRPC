## Generate JS code from xxx.proto file with protoc-gen-grpc (JS/.d.ts) (TS)

Generating TypeScript Definitions for CommonJS modules generated by protoc:

```
npm install protoc-gen-grpc -g

protoc-gen-grpc --js_out=import_style="commonjs,binary:./src/proto" --grpc_out=grpc_js:./src/proto --proto_path ./proto ./proto/*.proto
protoc-gen-grpc-ts --ts_out=grpc_js:./src/proto --proto_path ./proto ./proto/*.proto
```

#### Build:

```bash
npm run build

# CMD
mkdir .\build\proto
copy .\src\proto\* .\build\proto\

# PowerShell
New-Item -ItemType Directory -Force -Path .\build\proto\; Copy-Item -Path .\src\proto\* -Destination .\build\proto\ -Recurse

# bash
cp -r ./src/proto/* ./build/proto
```

#### Run:

```
npm run server
npm run client
```
